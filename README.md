# EchoBot - Bot de Telegram

Este es un bot de Telegram simple que repite ("echo") los mensajes que recibe.

## Funcionalidades

- Responde con un mensaje de bienvenida al comando `/start`.
- Repite todos los mensajes que se le envíen.
